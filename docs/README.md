# Documentation Map

This directory holds Vroom HR product contracts, architecture notes, and
decision records.

## Main Files

- `ARCHITECTURE.md`: architecture discovery and boundary rules.

## Folders

- `product/`: current product truth (module specs, guides, overview).
- `technical/`: cross-cutting technical references (migrations, seed data,
  error codes, shared infrastructure).
- `decisions/`: durable decisions and tradeoffs (ADRs).
- `templates/`: reusable documentation formats (e.g. decision records).
