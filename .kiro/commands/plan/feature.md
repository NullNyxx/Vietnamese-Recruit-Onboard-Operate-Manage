---
description: Create an implementation plan for a new feature
inclusion: manual
argument-hint: "[feature-name]"
---

## Arguments
FEATURE: $1 (required, feature name or description)

## Workflow
1. Analyze requirements and constraints
2. Research relevant patterns and prior art
3. Decompose into phases with deliverables
4. Identify dependencies and risks
5. Create plan in `plans/YYYYMMDD-HHmm-$1/`
6. Include success criteria and time estimates
