description: Design system architecture for a feature or service
inclusion: manual
argument-hint: "[system-name]"
---

## Arguments
SYSTEM: $1 (required, system or service name)

## Workflow
1. Gather requirements and constraints
2. Identify components and their responsibilities
3. Design data flow and communication patterns
4. Define API contracts between components
5. Document scaling and failure strategies
6. Create architecture diagram (Mermaid)
7. Save to design document
